# Mortgage-Risk-Analysis

# **Ипотечное кредитование: анализ первоначального взноса как фактора оценки риска**

## **Описание проекта**

Данный проект посвящен исследованию влияния первоначального взноса (ПВ) на платежеспособность клиентов в ипотечном кредитовании. Основная цель — проверить гипотезу о том, что отношение первоначального взноса к стоимости недвижимости является эффективным инструментом для ранжирования клиентов по уровню риска просрочки.

В рамках проекта решаются две ключевые задачи бизнеса:
- Обеспечение целевого уровня просрочки не выше 1%
- Максимизация уровня одобрения по заявкам на кредит

## **Ход исследования**

### **1. Анализ ретроспективных данных (main.ipynb)**

На первом этапе проведен анализ данных из файла `dataset.csv`, содержащего информацию о 7307 заявках:
- **application_id** — идентификатор заявки
- **pd** — вероятность просрочки (Probability of Default)
- **initinal_payment** — отношение первоначального взноса к стоимости недвижимости
- **delay** — факт просрочки (1 — просрочен, 0 — не просрочен)

В ходе анализа выполнены следующие шаги:
- Первичный анализ данных и расчет корреляции
- Бакетный анализ для оценки ранжирующей способности ПВ
- Пороговый анализ для определения оптимального значения ПВ
- Перекрестный анализ ПВ и pd для выявления дополнительной прогнозной силы признака

**Ключевой вывод:** гипотеза подтвердилась — при значении ПВ ≥ 31.6% формируется сегмент клиентов с уровнем просрочки менее 1%, охватывающий около 35% всех заявок.

### **2. Оценка эффекта на актуальных данных (task_3-sql-scripts/)**

Для оценки влияния предлагаемого правила на текущий поток заявок были написаны SQL-скрипты, работающие с данными из папки «БД». Скрипты позволяют:
- Рассчитать текущий уровень одобрения
- Определить количество заявок, подлежащих переквалификации
- Оценить новый уровень одобрения после внедрения правила
- Проанализировать эффект в разрезе источников поступления заявок

### **3. Разработка предложений и требований**

На основе проведенного анализа подготовлены:
- **Предложение по использованию ПВ в СПР** — описание того, как именно признак должен применяться в системе принятия решений
- **Бизнес-требование для разработчиков** — формальное описание правил, условий применения и ожидаемого поведения системы
- **Тестовые кейсы** — набор сценариев для проверки корректности реализации

## **Результаты**

- Подтверждена статистическая значимость первоначального взноса как фактора оценки риска
- Определен оптимальный порог ПВ = 31.6% для выделения низкорискового сегмента
- Доказана дополнительная прогнозная сила ПВ относительно существующей модели pd
- Рассчитан эффект от внедрения: рост уровня одобрения с 23.9% до 40.4% (на выборке из 1000 заявок)
