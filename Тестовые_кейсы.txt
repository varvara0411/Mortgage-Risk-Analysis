Тестовые кейсы

Тест-кейс 1. Срабатывание правила при ПВ выше порога.
Условие: request_type = FULL_APPLICATION, status = DECLINE, initinal_payment = 0.35.
Ожидаемый результат: правило применяется, решение корректируется в позитивную сторону.

Тест-кейс 2. Срабатывание правила при ПВ, равном порогу.
Условие: request_type = FULL_APPLICATION, status = DECLINE, initinal_payment = 0.316.
Ожидаемый результат: правило применяется, так как используется условие >= 0.316, решение корректируется.

Тест-кейс 3. Несрабатывание правила при ПВ ниже порога.
Условие: request_type = FULL_APPLICATION, status = DECLINE, initinal_payment = 0.315.
Ожидаемый результат: правило не применяется, решение остаётся без изменений.

Тест-кейс 4. Несрабатывание правила при отсутствии ПВ.
Условие: request_type = FULL_APPLICATION, status = DECLINE, initinal_payment is null.
Ожидаемый результат: правило не применяется, решение не меняется, система не падает с ошибкой.

Тест-кейс 5. Несрабатывание правила на другом этапе.
Условие: request_type = SHORT_APPLICATION, status = DECLINE, initinal_payment = 0.40.
Ожидаемый результат: правило не применяется, так как оно действует только на FULL_APPLICATION, решение не меняется.

Тест-кейс 6. Несрабатывание правила, если статус уже APPROVE.
Условие: request_type = FULL_APPLICATION, status = APPROVE, initinal_payment = 0.40.
Ожидаемый результат: правило не должно повторно менять уже положительное решение, статус остаётся APPROVE.

Тест-кейс 7. Блокировка правила стоп-фактором.
Условие: request_type = FULL_APPLICATION, status = DECLINE, initinal_payment = 0.40, по заявке присутствует обязательный стоп-фактор.
Ожидаемый результат: правило не применяется, решение остаётся отрицательным (применение правила заблокировано стоп-фактором)ю

Тест-кейс 8. Регрессионное тестирование.
Условие: подаётся набор заявок, для которых правило не должно быть применено.
Ожидаемый результат: результаты СПР полностью совпадают с результатами до внедрения бизнес-правила, новая бизнес-логика не влияет на неподходящие заявки.
